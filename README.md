# Constitutive Rhetoric Study

A multi-format public study of constitutive rhetoric in secessionist and restorationist movements, extending Maurice Charland's *Constitutive Rhetoric: The Case of the Peuple Québécois* (1987) across three cases: Québec sovereignty, Alberta's 2026 referendum, and the US "true America" framing.

Built with Jekyll, deployed on GitHub Pages (classic branch deploy from `main`, root folder). Push to `main` and the site rebuilds.

All writing is the author's working draft and subject to revision.

Live: https://lancerossiconsulting.github.io/constitutive-rhetoric-study/
